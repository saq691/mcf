产品介绍

帮派APP --- 国内首个移动端众包模式的企业自推广神器。它创造了企业移动营销推广新的模式。

本地商户使用过的推广手段形形色色，但其效果可能差强人意。举几个栗子：人肉发传单！… 80%的传单发出后就被随手扔掉；间歇低价团购！… 短期客流或许是上来了，但顾客每团一次，利润就流失一分，成本伤不起；发放实体会员卡！… 效果难以追踪，发出去的卡就如同泼出去的水。朋友圈发广告，瞬间被拉黑、被踢出，总是一个人在自嗨，发出去的广告就像泼出去的水。 为了解决这个难题帮派诞生了。针对小企业营销困境和市场痛点，以及各种环境因素，帮派为他们提供了一个更好的推广渠道，那就是众推直投。对于商家，可以在帮派商户端自主直投DM广告单；对于客户，可以在帮派派手端众包派发DM广告单，从而赚取佣金。

公司介绍
 

媒超风是一个新媒体公司，专注于移动数字化营销与品牌互动传播和基于大数据算法的移动广告匹配，是一家拥有10年专业网络营销经验和技术开发团队的网络运营商。目前公司已与腾讯、新浪等上千家网络媒体结成战略合作伙伴关系。

帮派是媒超风推出的国内首个众包广告平台。其首创P2P传播技术和大数据分析技术应用于传播领域。帮派app，一个基于大数据和自媒体的电子传单派发软件和众包传播平台，是一个移动端H5广告扩散神器。帮派是一个商家用户制作派发广告，普通自媒体用户查看广告、传播广告赚钱的平台。通过帮派商户端，商家用户自己的地盘自己做主，广告直投模式，自主定价，自主定量，自主选择投放区域、人群，真正做到了位置定向、人群定向、需求定向。真正解决了广告投放费用高投不起、没渠道、没权利，以及有曝光没流量、有流量没销量等问题。

FOREVER，媒超风将以让天下没有难做的推广为使命，为小而美发声到底，打造用户爱看的像春风一样的媒体。
